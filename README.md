# locai

Open a file, describe what to change, see the diff inline. Accept or reject changes line by line. Runs entirely on your machine with Ollama or LM Studio.

## Run

```bash
npm install
npm run tauri dev
```

## Requirements

- [Ollama](https://ollama.com) or [LM Studio](https://lmstudio.ai) running locally
- Rust 1.75+, Node.js 18+
